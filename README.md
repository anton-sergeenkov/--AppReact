# App React Guitar

## Описание проекта

### Компонент "Каталог товаров"
- Загрузка каталога товаров с тестового json-сервера по ajax запросу (axios)
- Отображения каталога товаров и детализации по товару

## Технологии
- React.js, React Router, Redux
- Webpack
- Axios
- HTML5, CSS3
